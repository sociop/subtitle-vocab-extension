# Subtitle Vocab Catcher

Расширение для Chrome: захватывает текущую фразу субтитров на плеерах киного и фликсбар,
переводит её через DeepL, показывает попап с фразой + переводом, где можно
кликнуть на слово, увидеть его перевод и одним кликом добавить в Anki
(через AnkiConnect).

Проще говоря, решение личной проблемы в изучении языка: быстрый перевод субтитров для быстрого перевода незнакомых слов.

## Установка расширения (пока не в Chrome Web Store)

1. Открой Chrome → `chrome://extensions`
2. Включи **Режим разработчика** (переключатель справа вверху)
3. Нажми **"Загрузить распакованное расширение"**
4. Укажи папку `subtitle-vocab-extension`

## Настройка

1. Клик по иконке расширения → откроется страница настроек
2. Впиши:
   - **DeepL API ключ** — из https://www.deepl.com/your-account/keys
   - **DeepL хост** — `https://api-free.deepl.com` для бесплатного ключа,
     `https://api.deepl.com` для платного (Pro)
   - **Колода Anki** — название колоды, куда падают карточки (например `Subtitles`)
   - **Тип карточки** — `Basic` подходит по умолчанию (поля Front/Back)
3. Жми "Сохранить"

## Требования для работы с Anki

- Anki должен быть **открыт** во время просмотра
- Установлен аддон **AnkiConnect** (код `2055492159` в Anki → Tools → Add-ons → Get Add-ons)
- В настройках AnkiConnect не нужно ничего менять, если и Anki, и браузер — на одной машине

## Как пользоваться

1. Включи фильм на плеере киного/фликсбар, включи английские субтитры
2. Когда на экране появится фраза, которую хочешь разобрать — жми `Ctrl+Shift+X`
3. Видео ставится на паузу, появляется попап:
   - Оригинал фразы (слова кликабельны)
   - Перевод фразы целиком
4. Клик по нужному слову → показывается перевод слова
5. Жми **"+ Добавить в Anki"** → карточка улетает в твою колоду
6. `Ctrl+Shift+X` ещё раз (или `Esc`) — закрыть попап, видео продолжает играть

## Возможные следующие шаги (для Claude Code)

- Добавить историю переводов внутри расширения (chrome.storage) — отдельный
  список слов за сессию просмотра
- Автоматически определять модель карточки в Anki и её поля через
  `AnkiConnect:modelFieldNames`, чтобы не зависеть от того, что поля называются
  Front/Back
- Добавить индикатор "слово уже есть в колоде" (через `findNotes`) чтобы не
  дублировать карточки
- Улучшить парсинг слова (сейчас просто убираются небуквенные символы —
  сокращения вроде `don't` могут теряться неверно)
